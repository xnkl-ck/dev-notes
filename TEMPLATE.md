# 笔记模板

这是一个通用的笔记模板，可以复制使用。

---

## 🐞 问题标题 / 📝 知识点标题

### 💬 发现场景 / 学习场景
简要描述在什么情况下遇到这个问题或学到这个知识点。

### ❌ 错误示例（如果适用）
```javascript
// 展示错误的代码或不好的做法
function badExample() {
  // ...
}
```

### ✅ 正确写法 / 解决方案
```javascript
// 展示正确的代码或好的做法
function goodExample() {
  // ...
}
```

### 🔍 原因分析（可选）
解释为什么会出现这个问题，或者这个方案为什么是好的。

### 💡 总结与延伸
- 关键要点 1
- 关键要点 2
- 相关知识点
- 延伸思考

### 📚 参考资料（可选）
- [相关文档链接](https://example.com)
- [博客文章](https://example.com)

### 🔖 标签
`#category` `#topic` `#keyword`
