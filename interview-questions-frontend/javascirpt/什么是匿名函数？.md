# **什么是匿名函数？**

## 前言

匿名函数是 JavaScript 中一个常见的函数形式，它没有绑定名称。匿名函数灵活且便于传递，常用于回调、立即执行函数和函数表达式。理解匿名函数的定义和使用，有助于写出更加简洁和灵活的代码，同时也是面试中常见的基础知识点。

## 匿名函数的定义及特点

匿名函数指没有名字的函数表达式。与具名函数不同，它没有函数名标识，通常作为值赋给变量或直接作为参数传递。

```plain
const add = function(a, b) {
  return a + b;
};
console.log(add(2, 3)); // 5
```

此处 `function(a, b) { ... }` 是匿名函数，被赋值给变量 `add`。

匿名函数的特点：

- 没有函数名，匿名
- 通常作为表达式使用
- 可以被赋值给变量、传递给函数或立即执行

## 匿名函数的常见使用场景

### 1. 回调函数

匿名函数多用于事件处理、异步操作中的回调。

```plain
setTimeout(function() {
  console.log('Timeout executed');
}, 1000);
```

### 2. 立即执行函数表达式（IIFE）

通过匿名函数立即执行，创建独立作用域，避免全局污染。

```plain
(function() {
  const temp = 'IIFE scope';
  console.log(temp);
})();
```

### 3. 赋值给变量或对象属性

将匿名函数赋值给变量或对象方法，动态定义行为。

```plain
const greet = function(name) {
  return 'Hello ' + name;
};
console.log(greet('Alice'));
```

### 4. 箭头函数形式的匿名函数

ES6 引入箭头函数，也经常用于写匿名函数，语法更加简洁。

```plain
const multiply = (x, y) => x * y;
console.log(multiply(2, 4)); // 8
```

## 匿名函数与具名函数的区别

- 具名函数有名称，便于调试和递归调用。
- 匿名函数没有名称，通常作为表达式使用。
- 具名函数声明会在代码加载时提升，匿名函数表达式不会提升。

## 面试回答

**问题：什么是匿名函数？**

回答示范：

匿名函数是没有函数名的函数表达式，通常作为变量的值、回调函数或者立即执行函数使用。它没有名称，使用时一般通过赋值给变量或者直接传递给其他函数。匿名函数灵活，适合动态创建函数和避免全局污染。现代 JavaScript 还使用箭头函数写匿名函数，语法更简洁。相比具名函数，匿名函数不具备函数声明提升，但更适合函数式编程。

## 总结

匿名函数是 JavaScript 中灵活且常用的函数形式，适合回调、事件处理、立即执行以及动态赋值等场景。理解匿名函数与具名函数的区别及其用法，有助于写出简洁、模块化的代码结构，同时对掌握 JavaScript 函数机制非常重要。
